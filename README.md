# Student Performance Prediction using Random Forest

## Overview

This project predicts whether a student is likely to **Pass** or **Fail** based on academic performance indicators using the Random Forest Classification algorithm. The model is trained on a sample student dataset and uses features such as study hours, attendance, assignments completed, and previous marks to make predictions.

The project demonstrates the complete machine learning workflow, including data preparation, model training, prediction, model evaluation, and visualization.

---

## Objectives

* Build a machine learning model to predict student performance.
* Encode categorical target values for model training.
* Train a Random Forest Classifier using student data.
* Predict the outcome for new student inputs.
* Evaluate the model using a confusion matrix.
* Analyze feature importance to understand the factors influencing predictions.

---

## Dataset

The project uses a sample dataset containing student academic information with the following attributes:

* Study Hours
* Attendance (%)
* Assignments Completed
* Previous Marks
* Result (Pass/Fail)

---

## Methodology

The project follows these steps:

* Created a student performance dataset using Pandas.
* Encoded the target variable using LabelEncoder.
* Selected relevant features for model training.
* Trained a Random Forest Classifier with multiple decision trees.
* Calculated the training accuracy of the model.
* Accepted user input to predict whether a student is likely to pass or fail.
* Evaluated the model using a confusion matrix.
* Visualized feature importance and one decision tree from the trained Random Forest.
